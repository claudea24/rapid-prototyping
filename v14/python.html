2.3 NumPy Patterns

▾

NumPy's ndarray enables vectorized operations — handing entire array operations to compiled C loops instead of Python's slow for-loop. Broadcasting lets you operate on arrays of different shapes without copying data. Use @ for matrix multiplication (dot product), * for element-wise product.

import numpy as np

X = np.array([[1, 2, 5],
              [1, 3, 8],
              [1, 5, 12]], dtype=float)
y = np.array([10, 15, 24], dtype=float)

# Closed-form OLS: β = (XᵀX)⁻¹ Xᵀy
beta = np.linalg.inv(X.T @ X) @ X.T @ y
print(beta)  # array([ 0.5,  0.5,  1.5])
Key Insight

When you see for i in range(len(arr)) with NumPy arrays, stop and ask: "Can I express this as a vectorized operation?" Almost always, the answer is yes.

Worked Example — Two Sum: Hash Map O(n)

For each element, compute the complement (target − num), check the dictionary in O(1), and either return or store the current element.

def two_sum(nums, target):
    seen = {}  # value -> index
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i

print(two_sum([2, 7, 11, 15], 9))  # [0, 1]

# Mutable default argument — the bug
def append_to(item, target=[]):
    target.append(item)
    return target

print(append_to(1))  # [1]
print(append_to(2))  # [1, 2] — SAME list!

# The fix: use None sentinel
def append_to(item, target=None):
    if target is None:
        target = []
    target.append(item)
    return target
