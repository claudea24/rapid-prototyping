Statistics

Descriptive stats, hypothesis testing, A/B tests, confidence intervals, and the Central Limit Theorem.

Descriptive Stats

Key
  • Mean — sensitive to outliers
  • Median — robust to outliers, better for skewed data
  • Mode — most frequent value
  • Variance — avg squared deviation from mean
  • Std Dev — sqrt(variance), same units as data
  • IQR — Q3 − Q1, used for outlier detection

Hypothesis Testing

Key
  • H₀ — null hypothesis (no effect / no difference)
  • H₁ — alternative hypothesis
  • p-value — P(data | H₀ true). Reject H₀ if p < α
  • Type I error — false positive (reject true H₀)
  • Type II error — false negative (fail to reject false H₀)
  • Power = 1 − P(Type II) — probability of detecting real effect

Common Tests

Key
  • z-test — known population variance, large n
  • t-test — unknown variance, small n
  • Paired t-test — before/after on same subjects
  • Chi-squared — categorical independence
  • ANOVA — compare means across 3+ groups
  • Mann-Whitney U — non-parametric alternative to t-test

A/B Testing

Tip
  1. Define metric and hypothesis
  2. Calculate sample size (power analysis)
  3. Randomize users into control / treatment
  4. Run for predetermined duration
  5. Analyze with appropriate test

Pitfalls: peeking, multiple comparisons, novelty effect, Simpson's paradox

Confidence Intervals

Formula
CI = x̄ ± z · (σ / √n)

A 95% CI means if we repeated sampling, 95% of intervals would contain the true parameter. Does NOT mean 95% probability the parameter is in this interval.

Central Limit Theorem

Key

Sample means approach a normal distribution as n increases, regardless of population distribution (given finite variance). Generally applies when n ≥ 30.

SE = σ / √n
