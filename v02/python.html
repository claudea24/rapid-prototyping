Data Structures Complexity

Key
  • list — append O(1), index O(1), search O(n), insert O(n)
  • dict — get/set O(1) avg, iteration O(n)
  • set — add/remove/lookup O(1) avg
  • deque — append/pop both ends O(1)
  • heap — push/pop O(log n), min O(1)

Pandas Essentials

Key 
# Grouping & aggregation
df.groupby('col')['val'].agg(['mean','sum'])

# Merging
df.merge(df2, on='id', how='left')

# Pivot table
df.pivot_table(values='rev', index='month', aggfunc='sum')

# Missing data
df.fillna(0) | df.dropna()

# Apply
df.apply(lambda x: x * 2)

NumPy Patterns

Tip 
import numpy as np

a = np.array([1,2,3])
np.mean(a) | np.std(a) | np.median(a)
np.dot(a, b)           # dot product
np.linalg.inv(M)      # matrix inverse
a[a > 2]              # boolean indexing
np.random.normal(0, 1, 1000)

Common Coding Patterns

Key
  • Two pointers — sorted array problems
  • Hash map — counting, grouping, lookup
  • Sliding window — subarray / substring
  • Sorting + greedy — optimization
  • defaultdict / Counter — frequency counts
from collections import Counter
c = Counter(words)
c.most_common(10)
#

String & List Tricks

Tip 
# List comprehension
[x**2 for x in range(10) if x % 2 == 0]

# Zip two lists
dict(zip(keys, values))

# Sort by custom key
sorted(data, key=lambda x: x[1], reverse=True)

Gotchas

Watch
  • Mutable defaults — never use def f(x=[])
  • Shallow copy — a = b[:] or copy.deepcopy()
  • Integer division — // truncates toward −inf
  • is vs == — is checks identity, == checks value
