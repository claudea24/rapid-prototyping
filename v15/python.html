Pandas Essentials

+75 XP ▼

The golden rule of Pandas: if you are writing a for-loop over rows, you are almost certainly doing it wrong.

GroupBy: Split-Apply-Combine — partition the DataFrame into groups, apply a function to each group independently, then stitch results back together.

import pandas as pd

df = pd.DataFrame({
    'region':  ['East', 'West', 'East', 'West', 'East'],
    'product': ['Widget', 'Gadget', 'Gadget', 'Widget', 'Widget'],
    'revenue': [120, 340, 210, 150, 95]
})

summary = df.groupby('region')['revenue'].agg(
    total_revenue='sum',
    avg_order='mean',
    num_transactions='count'
)
#         total_revenue  avg_order  num_transactions
# East              425     141.67                 3
# West              490     245.00                 2

NumPy Patterns

+75 XP ▼

Vectorization: NumPy sidesteps Python's per-element overhead by handing entire operations to compiled C loops. Speedups of 10–100× are typical.

Broadcasting: Arrays of different shapes can operate together. Dimensions are compared right to left — compatible if equal or one is 1.

* vs @: * is element-wise (Hadamard) product. @ is matrix multiplication (dot product).

import numpy as np

X = np.array([
    [1, 2, 5],
    [1, 3, 8],
    [1, 5, 12]
], dtype=float)

y = np.array([10, 15, 24], dtype=float)

# Closed-form OLS: beta = (X^T X)^-1 X^T y
XtX     = X.T @ X
XtX_inv = np.linalg.inv(XtX)
beta    = XtX_inv @ X.T @ y
# array([ 0.5,  0.5,  1.5])
# y ≈ 0.5 + 0.5·x1 + 1.5·x2
Key Insight
When you see for i in range(len(arr)) with NumPy arrays, stop and ask: "Can I express this as a vectorized operation?" Almost always, the answer is yes.

# Naive O(n²)
def two_sum_naive(nums, target):
    for i in range(len(nums)):
        for j in range(i + 1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]

# Hash map O(n)
def two_sum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i

print(two_sum([2, 7, 11, 15], 9))  # [0, 1]

Key Insight
The hash map pattern is the single most important tool in coding interviews. Whenever a brute-force uses a nested loop where the inner loop is searching, ask: "Can I replace that search with a dictionary lookup?"

Python Gotchas

+75 XP ▼

Mutable Default Arguments: Default values are evaluated once at function definition, not each call.

def append_to(item, target=[]):
    target.append(item)
    return target

print(append_to(1))   # [1]
print(append_to(2))   # [1, 2] — SAME list!

# Fix: use None sentinel
def append_to(item, target=None):
    if target is None:
        target = []
    target.append(item)
    return target

Shallow vs Deep Copy: Assignment copies references. Shallow copy creates new outer container but shares inner objects. Deep copy recursively duplicates everything.

Floor Division: -7 // 2 = -4 (rounds toward −∞, not toward zero).

is vs ==: == compares values; is compares identity (same object in memory).
