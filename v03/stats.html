Concept Cards
Common

Descriptive Stats

The fundamental measures that summarize and describe a dataset's characteristics.

  • Mean — arithmetic average of all values
  • Median — middle value when data is sorted
  • Mode — most frequently occurring value
  • Variance — average squared deviation from the mean
  • Std Dev — square root of variance; same units as data
  • IQR — Q3 − Q1; measures spread of the middle 50%
Rare

Hypothesis Testing

A structured framework for making data-driven decisions under uncertainty.

  • H0 (Null) — default assumption; no effect or difference
  • H1 (Alternative) — the claim you're testing for
  • p-value — probability of observing data at least as extreme, given H0 is true
  • Type I Error — rejecting H0 when it's actually true (false positive)
  • Type II Error — failing to reject H0 when it's false (false negative)
  • Power — 1 − P(Type II); ability to detect a real effect
Common

Common Tests

The statistical tests you'll encounter most often in interviews and practice.

  • z-test — known population variance, large n
  • t-test — unknown variance, compares one sample mean to a value
  • Paired t-test — compares means from the same group at different times
  • Chi-squared — tests independence between categorical variables
  • ANOVA — compares means across 3+ groups
  • Mann-Whitney — non-parametric alternative to the t-test
Legendary

A/B Testing

The gold standard for measuring causal impact in product experiments.

  • Define the hypothesis and choose a primary metric
  • Calculate sample size for desired power (≥80%) and significance (α=0.05)
  • Randomly assign users to control and treatment groups
  • Run the experiment for the pre-determined duration
  • Analyze results and check for statistical significance

Common pitfalls: peeking at results early, multiple comparisons without correction, novelty effects, Simpson's paradox, and underpowered tests.

Rare

Confidence Intervals

A range of values that likely contains the true population parameter.

CI = x̄ ± z * (σ / √n)

A 95% CI means that if you repeated the experiment many times, about 95% of the computed intervals would contain the true parameter. The width depends on confidence level, sample size, and variability.

Rare

Central Limit Theorem

Regardless of the population distribution, the distribution of sample means approaches a normal distribution as the sample size increases (typically n ≥ 30). This is why so many statistical tests assume normality.

SE = σ / √n

The standard error (SE) decreases as sample size grows, meaning larger samples give more precise estimates of the population mean.
