Probability Cheat Sheet

Rules, Bayes' theorem, distributions, expected value, and classic puzzles.

1 Probability Rules

P(A ∪ B) = P(A) + P(B) - P(A ∩ B) P(A ∩ B) = P(A) · P(B|A) P(Aᶜ) = 1 - P(A)
  • Independent: P(A ∩ B) = P(A) · P(B)
  • Mutually exclusive: P(A ∩ B) = 0

Conditional Probability

P(A|B) = P(A ∩ B) / P(B). Think of it as "restricting the universe to B, then finding A within it."

Example

Two dice rolled. P(sum=7 | first die=3)? Universe is {(3,1),(3,2),...,(3,6)} = 6 outcomes. Sum=7 needs second=4. P = 1/6.

2 Bayes' Theorem

P(A|B) = P(B|A) · P(A) / P(B)

Key example: 99% accurate test + 1% prevalence = only 50% PPV

Worked Example: Disease Testing

Prevalence: 1%. Test sensitivity: 99%. Test specificity: 99%.

P(Disease | Positive) = ?

P(Pos | Disease) * P(Disease)
-----------------------------------
P(Pos | Disease)*P(Disease) + P(Pos | No Disease)*P(No Disease)

= 0.99 * 0.01 / (0.99*0.01 + 0.01*0.99)
= 0.0099 / 0.0198
= 0.50 = 50%

Even with a 99% accurate test, a positive result is only 50/50 when the disease is rare. This is why screening tests need confirmation.

Normal: 68-95-99.7 rule (±1σ, ±2σ, ±3σ)

When to Use Which

Bernoulli/Binomial: coin flips, click/no-click, pass/fail. Poisson: # of arrivals, # of errors, rare events. Normal: heights, test scores, sample means (CLT). Exponential: wait times, time to failure.

Central Limit Theorem

Sample means are approximately normal for n ≥ 30, regardless of population distribution. This is why so many tests assume normality.

4 Expected Value

E[X] = Σ x · P(x) E[aX + bY] = aE[X] + bE[Y] (even if dependent!) Var(X+Y) = Var(X) + Var(Y) (only if independent)

Linearity of Expectation

This is the most powerful tool in probability. It works even for dependent variables. Classic application: expected number of fixed points in a random permutation.

Example: Coupon Collector

Expected draws to collect all n types: E = n(1 + 1/2 + 1/3 + ... + 1/n) = n · H(n). For n=10: about 29.3 draws. Uses linearity: each "phase" of collecting one new type is geometric.

Monty Hall

You pick door 1. Host opens door 3 (goat). Should you switch to door 2? YES. Your initial pick had 1/3 chance. The other two doors had 2/3 combined. Host revealing one goat concentrates that 2/3 onto the remaining door.
