Decision Framework

Start with logistic regression as a baseline (interpretable, fast). Move to tree-based models (RF, XGBoost) for nonlinear relationships. Use SVM for high-dimensional sparse data (text). KNN is rarely used in production but good for understanding instance-based learning.

Interview Tip

Always mention the bias-variance tradeoff when comparing models. Linear models = high bias, low variance. Complex models = low bias, high variance. Regularization bridges the gap.

When Accuracy Fails

With 99% negative class, a model predicting all-negative gets 99% accuracy but 0% recall. Always check class balance first.

3 Bias-Variance Tradeoff

  • High bias = underfit (train error high)
  • High variance = overfit (train-test gap)
  • L1 (Lasso) = sparse features (feature selection)
  • L2 (Ridge) = stable, shrinks all coefficients
  • Elastic Net = L1 + L2 combined

Diagnosing the Problem

Underfit: increase model complexity, add features, reduce regularization. Overfit: more data, regularization, fewer features, dropout, early stopping.

Regularization

# L1 drives coefficients to exactly 0 (feature selection)
Loss = MSE + lambda * sum(|w_i|)

# L2 shrinks coefficients toward 0 (but never exactly 0)
Loss = MSE + lambda * sum(w_i^2)

Handling Missing Values

Numerical: median (robust to outliers), mean (if symmetric). Categorical: mode or "Unknown" category. Always: add a binary "is_missing" indicator feature.

Target Encoding Safely

Target encoding replaces a category with the mean target value. To avoid leakage: use leave-one-out encoding or fit on training data only, then transform test data.

5 Cross-Validation

  • K-Fold: standard, K=5 or 10
  • Stratified: preserves class distribution
  • Time Series: never use future data (expanding window)
  • Pipeline: fit scaler/encoder inside CV to prevent leakage

Data Leakage Prevention

# WRONG: fit scaler on all data, then split
scaler.fit(X)
X_scaled = scaler.transform(X)
cross_val_score(model, X_scaled, y)

# RIGHT: use Pipeline (scaler fits inside each fold)
from sklearn.pipeline import Pipeline
pipe = Pipeline([('scaler', StandardScaler()),
                 ('model', LogisticRegression())])
cross_val_score(pipe, X, y)
Time Series CV

Never shuffle time series data. Use expanding or sliding window: train on months 1-3, test on 4. Then train on 1-4, test on 5. And so on.
