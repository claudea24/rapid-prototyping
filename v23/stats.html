Statistics Cheat Sheet

Central tendency, hypothesis testing, confidence intervals, and A/B testing.

1 Central Tendency

MeasureBest ForSensitive To
MeanSymmetric dataOutliers
MedianSkewed dataNothing (robust)
ModeCategoricalMultimodal data

Right-skewed: mean > median. Left-skewed: mean < median.

Spread Measures

Variance: average squared deviation from mean. Std Dev: square root of variance (same units as data). IQR: Q3 - Q1 (robust to outliers). Use IQR for skewed data, std dev for symmetric.

# Outlier detection with IQR
Q1, Q3 = np.percentile(data, [25, 75])
IQR = Q3 - Q1
lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR

2 Hypothesis Testing

H₀ → H₁ → α → Collect data → Test statistic → p-value → Conclude
  • p < α: reject H₀ (statistically significant)
  • Type I: false positive (reject true H₀) = α
  • Type II: false negative (keep false H₀) = β
  • Power = 1 - β (ability to detect real effect)

Common Misconceptions

p-value is NOT the probability that H₀ is true. It's the probability of seeing data this extreme IF H₀ were true. "Statistically significant" does not mean "practically important" -- always check effect size.

Increasing Power

Larger sample size, larger effect size, higher α (less strict), less noise in measurements.

Choosing the Right Test

Ask: (1) How many groups? (2) Continuous or categorical? (3) Paired or independent? (4) Normal distribution?

# Two-sample t-test
from scipy import stats
t_stat, p_val = stats.ttest_ind(group_a, group_b)

# Chi-squared test
observed = [[50, 30], [20, 40]]
chi2, p, dof, expected = stats.chi2_contingency(observed)

4 Confidence Intervals

CI = x̄ ± z* × σ / √n
  • 95% CI: z* = 1.96
  • Interpretation: 95% of such intervals contain the true parameter
  • NOT: "95% probability the true value is in this interval"
  • Width halves when n quadruples (√n in denominator)

CI vs p-value

A 95% CI that excludes 0 is equivalent to p < 0.05 (for testing whether the parameter differs from 0). CIs are more informative because they show effect size and precision, not just significance.

# Bootstrap confidence interval (non-parametric)
import numpy as np
boot_means = [np.mean(np.random.choice(data, len(data)))
              for _ in range(10000)]
ci_lower, ci_upper = np.percentile(boot_means, [2.5, 97.5])

5 A/B Testing Checklist

  • 1. Define primary metric + guardrails
  • 2. Power analysis for sample size
  • 3. Randomize at user level
  • 4. Run for fixed duration (no peeking!)
  • 5. Analyze with pre-registered test

Sample Size Formula

n = (z_α + z_β)² × 2σ² / δ²

Where δ is the minimum detectable effect. Doubling the required precision (halving δ) quadruples sample size.

Common Pitfalls

Peeking: checking results daily inflates false positives. Use sequential testing if you must peek. Simpson's paradox: overall trend reverses within segments. Network effects: users in treatment affect control (use cluster randomization).
