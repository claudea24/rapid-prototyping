2 Mutable Default Arguments Gotcha

▾

Default arguments are evaluated once at function definition time, not at each call. If the default is mutable (like a list), it is shared across all calls.

# Bug: shared mutable default
def add_item(item, lst=[]):
    lst.append(item)
    return lst

print(add_item(1))  # [1]
print(add_item(2))  # [1, 2] — not [2]!

# Fix: use None sentinel
def add_item(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst
Interview Insight

This is a classic Python gotcha. Always use None as the default and create the mutable object inside the function body.

3 NumPy & Matrix Operations

▾

NumPy provides efficient array operations. The @ operator (Python 3.5+) performs matrix multiplication.

import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Matrix multiplication
C = A @ B           # same as np.matmul(A, B)
# Element-wise multiplication
D = A * B           # Hadamard product

5 Comprehensions & Generators

▾

List comprehensions are Pythonic and fast. Generators use lazy evaluation for memory efficiency.

# List comprehension
squares = [x**2 for x in range(10) if x % 2 == 0]

# Generator (lazy, memory-efficient)
squares_gen = (x**2 for x in range(10_000_000))

# Dict comprehension
word_lens = {w: len(w) for w in ['hello', 'world']}
