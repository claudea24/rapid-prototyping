Statistics — Inference & Testing

Hypothesis testing, p-values, confidence intervals, A/B testing, and the Central Limit Theorem.

1 P-Values & Hypothesis Testing

▾

A p-value is the probability of observing data at least as extreme as what was observed, assuming the null hypothesis is true. It is not the probability that the hypothesis is true.

ConceptDefinition
H₀ (Null)No effect / no difference (status quo)
H₁ (Alternative)There is an effect / difference
p-valueP(data this extreme | H₀ true)
Significance level (α)Threshold for rejecting H₀ (typically 0.05)
Type I ErrorReject H₀ when it's true (false positive)
Type II ErrorFail to reject H₀ when it's false (false negative)
Common Misconception

A p-value of 0.03 does NOT mean "3% chance the null is true." It means: "If the null were true, there's a 3% chance of seeing data this extreme."

2 A/B Testing & Peeking

▾

In a properly run A/B test, you set a sample size in advance and check results once at the end. Peeking (checking daily) inflates the false positive rate dramatically.

ApproachFPR at α=5%Notes
Fixed-horizon (check once)5%Correct approach
Daily peeking~26-30%Massively inflated!
Sequential testing5% (controlled)Uses adjusted boundaries
Interview Insight

If asked "what happens if you check an A/B test every day?" the answer is: the false positive rate inflates to ~30%. Use sequential testing methods or commit to a fixed sample size.

3 Confidence Intervals

▾

A 95% confidence interval means: if we repeated the experiment many times, 95% of the intervals constructed this way would contain the true parameter value.

It does not mean "there's a 95% probability the true value is in this particular interval." The true value is fixed; it either is or isn't in the interval.

StatementCorrect?
"95% of similarly constructed intervals contain the true value"Correct
"There is a 95% probability the true mean is in this interval"Incorrect (frequentist)
"The interval is narrower with larger sample sizes"Correct (SE decreases)
Formula Reminder

CI = point estimate ± z* × SE. For 95% CI, z* = 1.96. Standard error SE = s / √n.

4 Central Limit Theorem

▾

The CLT states that the sampling distribution of the sample mean approaches a normal distribution as sample size increases, regardless of the population's distribution.

ConditionDetail
Sample sizen ≥ 30 is the common rule of thumb
IndependenceObservations must be independent
ResultMean of sampling dist = population mean
Standard errorSE = σ / √n
Why It Matters

CLT is the foundation of hypothesis testing and confidence intervals. It's why we can use normal-based tests even when the underlying data isn't normally distributed.
