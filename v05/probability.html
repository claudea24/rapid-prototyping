🧩 Combinations
Addition Rule
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
For mutually exclusive events: P(A ∪ B) = P(A) + P(B)
Multiplication Rule
P(A ∩ B) = P(A) · P(B|A)
If independent: P(A ∩ B) = P(A) · P(B)
Complement
P(A') = 1 - P(A)
Counting
nPr = n! / (n-r)! nCr = n! / (r! · (n-r)!)
Law of Large Numbers: As the number of trials increases, the sample mean converges to the expected value. This is why casinos always win in the long run.

Expected Value
E[X] = Σ x · P(x)
The "center of mass" of a distribution
Variance
Var(X) = E[X²] - (E[X])² Var(X) = Σ (x - μ)² · P(x)
Standard deviation: σ = √Var(X)
Binomial Distribution
P(X=k) = C(n,k) · pᵏ · (1-p)ⁿ⁻ᵏ E[X] = np, Var(X) = np(1-p)
Poisson Distribution
P(X=k) = (λᵏ · e⁻ᵛ) / k! E[X] = λ, Var(X) = λ
Normal Distribution
f(x) = (1/σ√2π) · e⁻½((x-μ)/σ)²
68-95-99.7 rule for 1, 2, 3 standard deviations
Key insight: Bernoulli is a single trial, Binomial sums n Bernoulli trials, Poisson approximates Binomial when n is large & p is small, and Normal approximates Binomial when np ≥ 5.

Bayes' Theorem
Bayes' Theorem
P(A|B) = P(B|A) · P(A) / P(B)
In Words
Posterior = Likelihood · Prior / Evidence
Update your beliefs when new evidence arrives
Law of Total Probability
P(B) = Σ P(B|Aᵢ) · P(Aᵢ)
Used to compute the denominator (Evidence) in Bayes
Medical test example: A 99% accurate test with 1% disease prevalence: P(sick|+) = (0.99 × 0.01) / (0.99 × 0.01 + 0.01 × 0.99) ≈ 50%. Base rate matters enormously!
