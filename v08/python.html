What is Python?

Python is a friendly programming language that's great for data science. It's like having a super smart assistant that can do math, analyze data, and even draw pictures!

Basic Code

Here's a simple Python program:

print("Hello, Data Scientist!") numbers = [1, 2, 3, 4, 5] total = sum(numbers) print(f"Sum: {total}")

This prints a message and calculates the sum of numbers!
