🧠 Algorithm Challenge

Implement a function to find duplicates in a list:

Write a function that returns all duplicate elements in a list.

def find_duplicates(nums):
    seen = set()
    duplicates = set()
    for num in nums:
        if num in seen:
            duplicates.add(num)
        else:
            seen.add(num)
    return list(duplicates)
