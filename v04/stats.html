Statistics

Hypothesis testing, distributions, and experiment design

Descriptive Stats

Key

Summarize and describe the main features of a dataset:

  • Mean — arithmetic average of all values
  • Median — middle value when sorted; robust to outliers
  • Mode — most frequently occurring value
  • Variance — average of squared deviations from the mean
  • Standard Deviation — square root of variance; same units as data
  • IQR — interquartile range (Q3 − Q1); measures spread of the middle 50%

Hypothesis Testing

Key

A framework for making statistical decisions using data:

  • H0 (Null Hypothesis) — the default assumption; no effect or difference
  • H1 (Alternative Hypothesis) — what we want to prove; there is an effect
  • p-value — probability of observing the data (or more extreme) if H0 is true
  • Type I Error — false positive; rejecting H0 when it's actually true (α)
  • Type II Error — false negative; failing to reject H0 when it's false (β)
  • Power — 1 − β; probability of correctly rejecting a false H0

Common Tests

Key

Choose the right test for your data and hypothesis:

  • z-test — known population variance, large sample (n > 30)
  • t-test — unknown variance, small sample; compares means
  • Paired t-test — compares means from the same group at two times
  • Chi-squared test — tests independence between categorical variables
  • ANOVA — compares means across 3+ groups simultaneously
  • Mann-Whitney U — non-parametric alternative to the t-test

A/B Testing

Tip

A structured approach to running experiments:

  1. Define your metric and hypothesis clearly
  2. Calculate the required sample size for statistical power
  3. Randomly assign users to control and treatment groups
  4. Run the experiment for the pre-determined duration
  5. Analyze results and check for statistical significance

Common pitfalls: peeking at results early, stopping too soon, novelty effects, Simpson's paradox, and not accounting for multiple comparisons.

Confidence Intervals

Key

A range of values likely to contain the true population parameter:

CI = x̄ ± z* · (σ / √n)

A 95% CI means: if we repeated the experiment many times, 95% of the computed intervals would contain the true parameter. The width depends on confidence level, variability (σ), and sample size (n). Larger samples yield narrower intervals.

Central Limit Theorem

Key

Regardless of the population distribution, the distribution of sample means approaches a normal distribution as sample size increases (typically n ≥ 30). This is why so many statistical methods assume normality.

SE = σ / √n

The standard error (SE) decreases as sample size grows, meaning larger samples give more precise estimates of the population mean.

🌱 Practice Questions

Q1: A p-value of 0.03 means...
  • There is a 3% probability of observing this data (or more extreme) if H0 is true
  • There is a 3% chance the null hypothesis is true
  • The treatment effect is 3%
Q2: The Central Limit Theorem states that...
  • The distribution of sample means approaches a normal distribution as sample size increases
  • All data is normally distributed if the sample is large enough
  • The mean of a sample always equals the population mean
