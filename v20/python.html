When to Choose What

list: ordered, need index access. dict: key-value mapping, fast lookups. set: uniqueness, membership testing. deque: BFS queues, sliding windows. heapq: top-K problems, priority queues.

# defaultdict avoids KeyError
from collections import defaultdict
freq = defaultdict(int)
for x in data:
    freq[x] += 1

# Counter for frequency counting
from collections import Counter
Counter("abracadabra").most_common(3)
# [('a', 5), ('b', 2), ('r', 2)]

2 Pandas Essentials

  • groupby = split-apply-combine
  • merge = SQL-style joins (on, how)
  • pivot_table = reshape long to wide
  • apply = row/col-wise custom function
  • loc = label-based, iloc = position-based

3 NumPy Essentials

  • * = element-wise multiply
  • @ = matrix multiply (dot product)
  • Vectorize: don't loop, use broadcasting
  • axis=0 = along rows (collapse rows), axis=1 = along cols

Broadcasting Rules

Arrays with different shapes are compatible when: dimensions are equal, or one of them is 1. NumPy stretches the size-1 dimension to match.

import numpy as np
a = np.array([[1,2,3],[4,5,6]])  # shape (2,3)
b = np.array([10,20,30])          # shape (3,) -> broadcasts to (2,3)
print(a + b)  # [[11,22,33],[14,25,36]]

# Vectorized vs loop (100x faster)
# BAD:  [x**2 for x in arr]
# GOOD: arr ** 2

Two Pointer Example

# Two Sum on sorted array
def two_sum_sorted(nums, target):
    l, r = 0, len(nums) - 1
    while l < r:
        s = nums[l] + nums[r]
        if s == target: return [l, r]
        elif s < target: l += 1
        else: r -= 1

Sliding Window Example

# Max sum of subarray of size k
def max_subarray_sum(nums, k):
    window = sum(nums[:k])
    best = window
    for i in range(k, len(nums)):
        window += nums[i] - nums[i-k]
        best = max(best, window)
    return best

5 Python Gotchas

  • Mutable default args: use None sentinel
  • Shallow copy shares inner objects
  • -7 // 2 = -4 (floors toward negative infinity)
  • is checks identity, == checks equality
  • [x]*3 creates 3 refs to same list

Mutable Default Trap

# BAD: list is shared across calls!
def append(val, lst=[]):
    lst.append(val)
    return lst

# GOOD: use None sentinel
def append(val, lst=None):
    if lst is None:
        lst = []
    lst.append(val)
    return lst

Copy Depth

import copy
a = [[1,2],[3,4]]
b = a.copy()         # shallow: b[0] is a[0]
c = copy.deepcopy(a) # deep: fully independent
