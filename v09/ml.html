Interactive Scenario

Model Selection Challenge

You have a dataset with 100k samples, 50 features, and class imbalance. Choose the right approach.

Recommended Approach:

  • Handle imbalance: SMOTE or class weights
  • Feature selection: Recursive feature elimination or feature importance
  • Model: Gradient boosting (XGBoost) with hyperparameter tuning
  • Evaluation: Stratified k-fold CV with F1-score and AUC
  • Production: Deploy via REST API with monitoring for drift
